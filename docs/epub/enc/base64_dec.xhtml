Encrypt.base64_decrypt

securefile.Encrypt.base64_decrypt(self, commit=False)

See base64_encrypt

Decrypt a cipher text into plain text using b64decode.

Example:

...
enc.open()
enc.base64_decrypt(commit=True)
enc.close()
Parameters

commit=False (bool [https://docs.python.org/3/library/functions.html#bool]) -- Save the change or not.

Returns

Decrypted Message or Plain Text.

Raises

None
