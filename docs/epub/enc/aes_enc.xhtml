Encrypt.aes_encrypt

securefile.Encrypt.aes_encrypt()

Encrypt a plain text with AES algorithm. ( aes_algorithm )

Example:

from securefile import Encrypt
from securefile.keyset import AES_KEY
aes_key = AES_KEY.genrate('700102030405060708090a0b0c0d0e0f')
enc = Encrypt('test.txt')
enc.open()
enc.aes_encrypt(aes_key)
enc.close()
Parameters

  • key (AES_KEY) -- Encryption key.

  • commit=False (bool [https://docs.python.org/3/library/functions.html#bool]) -- Save the change or not.

Returns

Encrypted Message or Cipher Text.

Raises

  • IndexError [https://docs.python.org/3/library/exceptions.html#IndexError] -- Word size too large to encrypt with AES algorithm.

  • ValueError [https://docs.python.org/3/library/exceptions.html#ValueError] -- No a formetted text

Note

Most sutable after RSA or DES Algorithm
