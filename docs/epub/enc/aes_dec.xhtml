Encrypt.aes_decrypt

securefile.Encrypt.aes_decrypt()

Decrypt a cipher text into plain text. ( aes_encrypt )

Example:

...
enc.open()
enc.aes_decrypt(des_key, commit=True)
enc.close()
Parameters

  • key (AES_KEY) -- Encryption key.

  • commit=False (bool [https://docs.python.org/3/library/functions.html#bool]) -- Save the change or not.

Returns

Decrypted Message or Plain Text.

Raises

ValueError - Not a formetted text.
